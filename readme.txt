использовано: 
-react (https://ru.reactjs.org/)
-react-bootstrap (https://react-bootstrap.github.io/)
-json-server (https://github.com/typicode/json-server)
-axios (https://github.com/axios/axios)
-input-mask (https://www.npmjs.com/package/react-input-mask)

Для запуска: установить nodejs, npm
Из корневоой директории проекта установить зависимости коммандой npm install
+ установить json-server (файл бд уже создан и находиться в корне проекта bd.json) (npm install -g json-server)
+ сервер запустить на 3001 порту, либо поменять соответствующую переменнуюю в components/students.js.  (json-server --watch db.json --port 3001)
запустить коммандой npm start
